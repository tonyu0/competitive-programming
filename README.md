# competitive-programming

C++/Rust/Haskell/Assembly programs can be executed through Makefile for now, using simple but enough way for competitive programming.

## 1. Build the workspace
```sh
make build
```

## 2. Start the workspace
```sh
make start
```

## 3. Run programs
```sh
# Run a C++ program
make cpp-run file=program.cpp

# Run a Rust program
make rs-run file=program.rs

# Run a Haskell program
make hs-run file=program.hs

# Run a Assembly program
make asm-run file=program.asm
```

## 4. Enter the workspace
```sh
make shell
```

## 5. Stop the workspace
```sh
make stop
```
